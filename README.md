# spring-angular-rest-demo
Spring boot 2.0.3 + Angular6 + OAuth2 구현

## 적용
Spring Boot JPA
Spring Boot Security - OAuth2
H2 Database (inner db)

Angular6
Angular Material

## 설치 및 시작 방법

### Angular 설치
```
cd client
npm install
```
### Angular 시작 방법
```
cd client
npm start
```
localhost:4200 으로 시작됨

### Spring Boot 시작 방법
```
Ide 에서 gradle bootRun
```
localhost:8080 으로 시작됨
